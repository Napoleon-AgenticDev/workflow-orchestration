# Plan Specification (v2.0)

**Product:** Agent Alchemy (agent-alchemy-dev)  
**Focus:** Execute recommendations from SITE-CONTENT-ANALYSIS.md  
**Goal:** Transform agentalchemy.dev into a high-converting, SEO-ready product site

---

## 1. Strategic Objectives

1. Tell the **“Why Agent Alchemy Exists”** story (origin + problem + solution).  
2. Map messaging + CTAs to **4 personas** (Engineer, Architect, CTO/EM, Indie Founder).  
3. Add **trust & proof** (comparison table, testimonials, ROI calculator, social proof).  
4. Provide **low-friction onboarding** (starter kit, demo video, get-started walkthrough).  
5. Build an **SEO content hub** (pillar pages + blog + structured data).  
6. Measure everything (GA4 events, persona-tagged CTAs, heatmaps).

## 4. Campaign Sequencing

### Phase 1 – Foundation (Week 1)
- Launch new site sections (Origin, Comparison, Social Proof, Get Started, FAQ v2).  
- Implement meta tags, structured data, GA4 events.  
- Publish SITE-CONTENT-ANALYSIS summary post (internal enablement).

### Phase 2 – Trust & Conversion (Weeks 2–3)
- Collect + add 3 testimonials + case snippets.  
- Release ROI calculator + free starter kit + exit-intent offer.  
- Launch persona-specific CTAs + email automation.

### Phase 3 – SEO Engine (Weeks 4–5)
- Publish 3 blog pillars + case study landing pages.  
- Create /for-engineers, /for-architects, /for-ctos, /for-indie pages.  
- Produce 5-minute “origin + demo” video.

### Phase 4 – Optimization (Week 6+)
- A/B test hero copy + CTA text.  
- Analyze persona conversion data → iterate.  
- Add additional case studies, webinars, community proof.
